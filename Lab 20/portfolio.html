Below are links to the GitHub repos for some of the projects we've worked on:

  • MVC Countries

    Asks the user to select a country, then displays the name of the country, continent, and colors of the country's flag.

  • Rock, Paper, Scissors Lab

    Asks the user to select an opponent to play Rock, Paper, Scissors against and tracks the score for each player.

  • Movie Database

    Search for a movie by genre and the console will list related movies.

  • Alexandria Memorial Library Collaboration

    Library Database Console App that allows you to search for a book by title, author, or ISBN or display a list of all available books as well as donate a book or burn the library.
